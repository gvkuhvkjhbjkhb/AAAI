# HetToM Round-4 最终实验结论

> 实验环境：Lab RTX 5090 (32GB)，4-bit 量化本地推理
> 模型：Qwen2.5-7B-Instruct (Alibaba) + GLM-4-9B-0414 (Zhipu) — 真架构异质性
> 博弈：Stag Hunt (纯协调) + Battle of the Sexes (偏好冲突) + Chicken (反协调)
> 种子：Stag Hunt n=4, BoS/Chicken n=3

## 一、最核心的发现：方法的有效性是博弈结构条件函数

### 跨博弈回报矩阵

|---|---|---|---|---|

## 二、三个博弈的三种不同模式

### 模式 1：纯协调博弈 (Stag Hunt) — 对齐有效，多样性是代价

- 7B/9B 模型几乎"解决"了协调：多个方法达到 3.000 完美协作
- 所有对齐机制（ToM/talk/gating）都将多样性压到 0
- DP-gating 保留多样性(0.365)但付出回报代价(2.733 vs 3.000)
- **结论**：多样性-对齐张力存在，但博弈太简单，方法间区分度低

### 模式 2：偏好冲突博弈 (BoS) — 没有方法能解决偏好分歧

- **没有任何方法显著超越基线** (所有 p > 0.05)
- 异质性反而有害：het_notom 1.700 < hom_notom 2.133 (Δ=-0.433)
- het_tom_talk 是唯一正向的方法 (+0.247)，说明 ToM+talk 有助于均衡选择
- DP-gating 在这里**有害** (1.600)：保留多样性=保留分歧=更多错配
- ToM 准确率从 Stag Hunt 的 0.98 降到 0.78 — 偏好冲突让意图推理变难
- **结论**：偏好冲突是 LLM 协作的真瓶颈，现有机制无法解决

### 模式 3：反协调博弈 (Chicken) — 多样性是有利的

- 异质性单独就极强：het_notom 3.093 >> hom_notom 1.960 (+1.133)
- **DP-gating 在此博弈表现最优**：3.087 回报 + 0.442 多样性 — 同时实现了高多样性+高回报！
- 常规门控反而有害：2.573 < het_notom 3.093 — 强制对齐在反协调中是错的
- het_tom 2.507 < het_notom 3.093 — ToM 也有害，因为反协调需要"做不同的事"而非"推理对方做什么"
- **结论**：在反协调博弈中，多样性-对齐张力反转了——多样性本身就有利

## 三、模型规模的发现 (3B/1.5B vs 7B/9B)

| Cell | 3B/1.5B | 7B/9B | 变化 | 倍率 |
|---|---|---|---|---|
| hom_notom | 2.325 | 2.315 | -0.010 | 1.0x |
| **het_notom** | **0.549** | **2.735** | **+2.186** | **5.0x** |
| het_tom | 1.245 | 2.978 | +1.732 | 2.4x |
| het_notom_talk | 1.776 | 3.000 | +1.224 | 1.7x |
| het_tom_talk | 0.848 | 3.000 | +2.152 | 3.5x |
| het_gated_atom_talk | 1.734 | 3.000 | +1.266 | 1.7x |

**最震撼的数据点**：het_notom 回报从 0.549 跃升到 2.735 (5 倍)。Round-3 的"异质性摧毁协作"完全是 3B/1.5B 模型能力不足的假象。

Round-3 的三个"机制级突破"全部是解决弱模型制造的问题：
- "信念-信号干扰" → 7B/9B 下消失 (het_tom_talk = het_tom = 3.000)
- "A-ToM 从无效到有效" → 7B/9B 下增益不可区分
- "门控仲裁必需" → 7B/9B 下门控不再是必需品

## 四、AAAI 投稿评估

### 4.1 可用的故事资产

| 资产 | 强度 | 说明 |
|---|---|---|
| 跨博弈方法分化 | ★★★★ | 方法排名随博弈结构翻转 — 这是核心发现 |
| 3B vs 7B 对比 | ★★★★★ | 5x 跃升，证明弱模型结论是假象 — 最有冲击力 |
| DP-gating 在 Chicken 上最优 | ★★★ | 唯一同时高多样性+高回报的数据点 |
| BoS 上无方法有效 | ★★★ | 诚实负面结果，揭示 LLM 协作真瓶颈 |
| 多样性-对齐张力的结构依赖性 | ★★★★ | 张力不是universal的，是game-structure dependent |

### 4.2 最优论文定位

**标题方向**："When Does Heterogeneity Help LLM Agent Cooperation? A Scale- and Structure-Dependent Analysis"

**核心主张**：
1. **模型规模发现**：弱模型(3B)的"协作失败"是能力假象，7B/9B 下消失 (5x 跃升)
2. **博弈结构发现**：方法有效性是博弈结构的条件函数 — 没有universal最优方法
3. **多样性-对齐张力的结构依赖性**：
   - 纯协调(Stag Hunt)：张力存在（对齐压缩多样性）
   - 偏好冲突(BoS)：张力加剧（多样性=分歧=有害）
   - 反协调(Chicken)：张力反转（多样性本身有利）
4. **方法贡献**：DP-gating 是唯一能根据博弈结构自适应调节多样性保留的方法

### 4.4 致命弱点

1. **n=3 seeds (BoS/Chicken)**：Mann-Whitney 最小 p=0.0765，所有比较不显著。需扩到 n=8 达 p<0.05。
2. **DP-gating 不是universal最优**：在 BoS 上有害(1.600)，在 Stag Hunt 上有代价(2.733)。审稿人会问"什么时候该用你的方法？"
3. **缺乏 Layer-2 MARL 迁移**：纯 LLM-as-agent，无 RL 训练，RL 成分弱
4. **BoS 上所有方法失败**：虽然是诚实的负面结果，但"我的方法解决不了最难的问题"不是卖点
